section ‚ÄπFarkas Coefficients via the Simplex Algorithm of Duterte and de~Moura‚Ä∫

text ‚ÄπLet $c_1,\dots,c_n$ be a finite list of linear inequalities. 
  Let $C$ be a list of pairs $(r_i,c_i)$ where $r_i$ is a rational number.
  We say that $C$ is a list of \emph{Farkas coefficients} if
  the sum of all products $r_i \cdot c_i$ results in an inequality that is 
  trivially unsatisfiable.

  Farkas' Lemma  
  states that a finite set of non-strict linear inequalities is
  unsatisfiable if and only if Farkas coefficients exist. We will prove this lemma
  with the help of the simplex algorithm of Dutertre and de~Moura's.

  Note that the simplex implementation works on four layers, and we will formulate and prove 
  a variant of Farkas' Lemma for each of these layers.‚Ä∫

subsection ‚ÄπLinear Inequalities‚Ä∫

text ‚ÄπBoth Farkas' Lemma and Motzkin's Transposition Theorem require linear combinations 
  of inequalities. To this end we define one type that permits strict and non-strict 
  inequalities which are always of the form ``polynomial R constant'' where R is either 
  $\leq$ or $<$. On this type we can then define a commutative monoid.‚Ä∫

text ‚ÄπA type for the two relations: less-or-equal and less-than.‚Ä∫

subsection ‚ÄπFarkas' Lemma on Layer 4‚Ä∫

text ‚ÄπOn layer 4 the algorithm works on a state containing a tableau, atoms (or bounds), 
  an assignment and a satisfiability flag. Only non-strict inequalities appear at this level.
  In order to even state a variant of Farkas' Lemma on layer 4, we 
  need conversions from atoms to non-strict constraints and then further
  to linear inequalities of type @{type le_constraint}. 
  The latter conversion is a partial operation, since non-strict constraints
  of type @{type ns_constraint} permit greater-or-equal constraints, whereas @{type le_constraint}
  allows only less-or-equal.‚Ä∫

text ‚ÄπThe advantage of first going via @{type ns_constraint} is that this type permits a multiplication
  with arbitrary rational numbers (the direction of the inequality must be flipped when
  multiplying by a negative number, which is not possible with @{type le_constraint}).‚Ä∫

subsection ‚ÄπFarkas' Lemma on Layer 2‚Ä∫

text ‚ÄπThe main difference between layers 2 and 3 is the introduction of slack-variables in layer 3

  slack-variables are used) can be converted into Farkas coefficients for layer 2 (before the
  preprocessing).‚Ä∫


text ‚ÄπWe also need to adapt the previos notion of Farkas coefficients, which was used in 
  @{const farkas_coefficients_atoms_tableau}, for layer~2. At layer 3, Farkas coefficients
  are the coefficients in a linear combination of atoms that evaluates to an inequality of
  the form $p \leq c$, where $p$ is a linear polynomial, $c < 0$, and $t \models p = 0$ holds.
  At layer 2, the atoms are replaced by non-strict constraints where the left-hand side is a
  polynomial in the original variables, but the corresponding linear combination (with Farkas
  coefficients) evaluates directly to the inequality $0 \leq c$, with $c < 0$. The implication

  needed, since $p$ is $0$. Thus, the statement defining Farkas coefficients must be changed
  accordingly.‚Ä∫

lemma lec_of_constraint:

instantiation constraint :: scaleRat
begin
fun scaleRat_constraint :: "rat ‚áí constraint ‚áí constraint" where
  "scaleRat_constraint r cc = (if r = 0 then LEQ 0 0 else 
  (case cc of 
    LEQ p c ‚áí
     (if (r < 0) then GEQ (r *R p) (r *R c) else LEQ (r *R p) (r *R c))
  | LT p c ‚áí
     (if (r < 0) then GT (r *R p) (r *R c) else LT (r *R p) (r *R c))
  | GEQ p c ‚áí 
    (if (r > 0) then GEQ (r *R p) (r *R c) else LEQ (r *R p) (r *R c))
  | GT p c ‚áí 
    (if (r > 0) then GT (r *R p) (r *R c) else LT (r *R p) (r *R c))
  | LTPP p q ‚áí
     (if (r < 0) then GT (r *R (p - q)) 0 else LT (r *R (p - q)) 0)
  | LEQPP p q ‚áí
     (if (r < 0) then GEQ (r *R (p - q)) 0 else LEQ (r *R (p - q)) 0)
  | GTPP p q ‚áí
     (if (r > 0) then GT (r *R (p - q)) 0 else LT (r *R (p - q)) 0)
  | GEQPP p q ‚áí
     (if (r > 0) then GEQ (r *R (p - q)) 0 else LEQ (r *R (p - q)) 0)

subsection ‚ÄπFarkas' Lemma on Delta-Rationals‚Ä∫

text ‚ÄπWe start with Lemma~2 of \cite{Bromberger2017}, a
  variant of Farkas' Lemma for delta-rationals. To be more precise, it states
  that a set of non-strict inequalities over delta-rationals is unsatisfiable
  if and only if there is a linear combination of the inequalities that results
  in a trivial unsatisfiable constraint $0 < const$ for some negative constant $const$.
  We can easily prove this statement via the lemma @{thm [source] farkas_coefficients_ns} 
  and some conversions between the 
  different constraint types.‚Ä∫

lemma Farkas'_Lemma_Delta_Rationals: fixes cs :: "QDelta le_constraint set"
  assumes only_non_strict: "lec_rel ` cs ‚äÜ {Leq_Rel}"  
    and fin: "finite cs" 
  shows "(‚àÑ v. ‚àÄ c ‚àà cs. v ‚ä®‚á©l‚á©e c) ‚ü∑ 
       (‚àÉ C const. (‚àÄ (r, c) ‚àà set C. r > 0 ‚àß c ‚àà cs)
         ‚àß (‚àë (r,c) ‚Üê C. Leqc (r *R lec_poly c) (r *R lec_const c)) = Leqc 0 const
         ‚àß const < 0)" 
    (is "?lhs = ?rhs")
proof -
  {
    fix c
    assume "c ‚àà cs" 
    with only_non_strict have "lec_rel c = Leq_Rel" by auto
    then have "‚àÉ p const. c = Leqc p const" by (cases c, auto)
  } note leqc = this
  let ?to_ns = "Œª c. LEQ_ns (lec_poly c) (lec_const c)" 
  let ?ns = "?to_ns ` cs" 
  from fin have fin: "finite ?ns" by auto
  have "v ‚ä®‚á©n‚á©s‚á©s ?ns ‚ü∑ (‚àÄ c ‚àà cs. v ‚ä®‚á©l‚á©e c)" for v using leqc by fastforce
  hence "?lhs = (‚àÑ v. v ‚ä®‚á©n‚á©s‚á©s ?ns)" by simp
  also have "‚Ä¶ = (‚àÉC. farkas_coefficients_ns ?ns C)" unfolding farkas_coefficients_ns[OF fin] ..
  also have "‚Ä¶ = ?rhs" 
  proof
    assume "‚àÉ C. farkas_coefficients_ns ?ns C" 
    then obtain C const where is_leq: "‚àÄ (s, n) ‚àà set C. n ‚àà ?ns ‚àß is_leq_ns (s *R n) ‚àß s ‚â† 0" 
      and sum: "(‚àë(s, n)‚ÜêC. lec_of_nsc (s *R n)) = Leqc 0 const"
      and c0: "const < 0" unfolding farkas_coefficients_ns_def by blast
    let ?C = "map (Œª (s,n). (s,lec_of_nsc n)) C" 
    show ?rhs 
    proof (intro exI[of _ ?C] exI[of _ const] conjI c0, unfold sum[symmetric] map_map o_def set_map, 
        intro ballI, clarify)
      {
        fix s n 
        assume sn: "(s, n) ‚àà set C" 
        with is_leq
        have n_ns: "n ‚àà ?ns" and is_leq: "is_leq_ns (s *R n)" "s ‚â† 0" by force+
        from n_ns obtain c where c: "c ‚àà cs" and n: "n = LEQ_ns (lec_poly c) (lec_const c)" by auto
        with leqc[OF c] obtain p d where cs: "Leqc p d ‚àà cs" and n: "n = LEQ_ns p d" by auto
        from is_leq[unfolded n] have s0: "s > 0" by (auto split: if_splits) 
        let ?n = "lec_of_nsc n" 
        from cs n have mem: "?n ‚àà cs" by auto
        show "0 < s ‚àß ?n ‚àà cs" using s0 mem by blast        
        have "Leqc (s *R lec_poly ?n) (s *R lec_const ?n) = lec_of_nsc (s *R n)" 
          unfolding n using s0 by simp
      } note id = this
      show "(‚àëx‚ÜêC. case case x of (s, n) ‚áí (s, lec_of_nsc n) of
             (r, c) ‚áí Leqc (r *R lec_poly c) (r *R lec_const c)) =
             (‚àë(s, n)‚ÜêC. lec_of_nsc (s *R n))" (is "sum_list (map ?f C) = sum_list (map ?g C)")
      proof (rule arg_cong[of _ _ sum_list], rule map_cong[OF refl])
        fix pair
        assume mem: "pair ‚àà set C" 
        then obtain s n where pair: "pair = (s,n)" by force
        show "?f pair = ?g pair" unfolding pair split using id[OF mem[unfolded pair]] .
      qed
    qed
  next
    assume ?rhs
    then obtain C const 
      where C: "‚ãÄ r c. (r,c) ‚àà set C ‚üπ 0 < r ‚àß c ‚àà cs" 
        and sum: "(‚àë(r, c)‚ÜêC. Leqc (r *R lec_poly c) (r *R lec_const c)) = Leqc 0 const"
        and const: "const < 0" 
      by blast
    let ?C = "map (Œª (r,c). (r, ?to_ns c)) C"  
    show "‚àÉ C. farkas_coefficients_ns ?ns C" unfolding farkas_coefficients_ns_def
    proof (intro exI[of _ ?C] exI[of _ const] conjI const, unfold sum[symmetric])
      show "‚àÄ(s, n)‚ààset ?C. n ‚àà ?ns ‚àß is_leq_ns (s *R n) ‚àß s ‚â† 0" using C by fastforce
      show "(‚àë(s, n)‚Üê?C. lec_of_nsc (s *R n)) 
        = (‚àë(r, c)‚ÜêC. Leqc (r *R lec_poly c) (r *R lec_const c))"
        unfolding map_map o_def
        by (rule arg_cong[of _ _ sum_list], rule map_cong[OF refl], insert C, force)

  qed
  finally show ?thesis .
qed



subsection ‚ÄπMotzkin's Transposition Theorem or the Kuhn-Fourier Theorem‚Ä∫

text ‚ÄπNext, we prove a generalization of Farkas' Lemma that permits arbitrary combinations
  of strict and non-strict inequalities: Motzkin's Transposition Theorem
  which is also known as the Kuhn--Fourier Theorem.

  The proof is mainly based on the lemma @{thm [source] farkas_coefficients}, 
  again requiring conversions between constraint types.‚Ä∫

theorem Motzkin's_transposition_theorem: fixes cs :: "rat le_constraint set"
  assumes fin: "finite cs" 
  shows "(‚àÑ v. ‚àÄ c ‚àà cs. v ‚ä®‚á©l‚á©e c) ‚ü∑ 
       (‚àÉ C const rel. (‚àÄ (r, c) ‚àà set C. r > 0 ‚àß c ‚àà cs)
         ‚àß (‚àë (r,c) ‚Üê C. Le_Constraint (lec_rel c) (r *R lec_poly c) (r *R lec_const c)) 
               = Le_Constraint rel 0 const
         ‚àß (rel = Leq_Rel ‚àß const < 0 ‚à® rel = Lt_Rel ‚àß const ‚â§ 0))" 
    (is "?lhs = ?rhs")
proof -
  let ?to_cs = "Œª c. (case lec_rel c of Leq_Rel ‚áí LEQ | _ ‚áí LT) (lec_poly c) (lec_const c)" 
  have to_cs: "v ‚ä®‚á©c ?to_cs c ‚ü∑ v ‚ä®‚á©l‚á©e c" for v c by (cases c, cases "lec_rel c", auto)
  let ?cs = "?to_cs ` cs" 
  from fin have fin: "finite ?cs" by auto
  have "v ‚ä®‚á©c‚á©s ?cs ‚ü∑ (‚àÄ c ‚àà cs. v ‚ä®‚á©l‚á©e c)" for v using to_cs by auto
  hence "?lhs = (‚àÑ v. v ‚ä®‚á©c‚á©s ?cs)" by simp
  also have "‚Ä¶ = (‚àÉC. farkas_coefficients ?cs C)" unfolding farkas_coefficients[OF fin] ..
  also have "‚Ä¶ = ?rhs" 
  proof
    assume "‚àÉ C. farkas_coefficients ?cs C" 
    then obtain C const rel where is_leq: "‚àÄ (s, n) ‚àà set C. n ‚àà ?cs ‚àß is_le (s *R n) ‚àß s ‚â† 0" 
      and sum: "(‚àë(s, n)‚ÜêC. lec_of_constraint (s *R n)) = Le_Constraint rel 0 const"
      and c0: "(rel = Leq_Rel ‚àß const < 0 ‚à® rel = Lt_Rel ‚àß const ‚â§ 0)" 
      unfolding farkas_coefficients_def by blast
    let ?C = "map (Œª (s,n). (s,lec_of_constraint n)) C" 
    show ?rhs 
    proof (intro exI[of _ ?C] exI[of _ const] exI[of _ rel] conjI c0, unfold map_map o_def set_map sum[symmetric], 
        intro ballI, clarify)
      {
        fix s n 
        assume sn: "(s, n) ‚àà set C" 
        with is_leq 
        have n_ns: "n ‚àà ?cs" and is_leq: "is_le (s *R n)" and s0: "s ‚â† 0" by force+
        from n_ns obtain c where c: "c ‚àà cs" and n: "n = ?to_cs c" by auto
        from is_leq[unfolded n] have "s ‚â• 0" by (cases "lec_rel c", auto split: if_splits) 
        with s0 have s0: "s > 0" by auto
        let ?c = "lec_of_constraint n" 
        from c n have mem: "?c ‚àà cs" by (cases c, cases "lec_rel c", auto)
        show "0 < s ‚àß ?c ‚àà cs" using s0 mem by blast        
        have "lec_of_constraint (s *R n) = Le_Constraint (lec_rel ?c) (s *R lec_poly ?c) (s *R lec_const ?c)" 
          unfolding n using s0 by (cases c, cases "lec_rel c", auto)
      } note id = this
      show "(‚àëx‚ÜêC. case case x of (s, n) ‚áí (s, lec_of_constraint n) of
             (r, c) ‚áí Le_Constraint (lec_rel c) (r *R lec_poly c) (r *R lec_const c)) =
            (‚àë(s, n)‚ÜêC. lec_of_constraint (s *R n))"

      proof (rule arg_cong[of _ _ sum_list], rule map_cong[OF refl])
        fix pair
        assume mem: "pair ‚àà set C"

      qed
    qed
  next
    assume ?rhs
    then obtain C const rel 
      where C: "‚ãÄ r c. (r,c) ‚àà set C ‚üπ 0 < r ‚àß c ‚àà cs" 
        and sum: "(‚àë(r, c)‚ÜêC. Le_Constraint (lec_rel c) (r *R lec_poly c) (r *R lec_const c)) 
           = Le_Constraint rel 0 const"
        and const: "rel = Leq_Rel ‚àß const < 0 ‚à® rel = Lt_Rel ‚àß const ‚â§ 0" 
      by blast
    let ?C = "map (Œª (r,c). (r, ?to_cs c)) C"  
    show "‚àÉ C. farkas_coefficients ?cs C" unfolding farkas_coefficients_def
    proof (intro exI[of _ ?C] exI[of _ const] exI[of _ rel] conjI const, unfold sum[symmetric])
      show "‚àÄ(s, n)‚ààset ?C. n ‚àà ?cs ‚àß is_le (s *R n) ‚àß s ‚â† 0" using C by (fastforce split: le_rel.splits) 
      show "(‚àë(s, n)‚Üê?C. lec_of_constraint (s *R n))

subsection ‚ÄπFarkas' Lemma‚Ä∫

text ‚ÄπFinally we derive the commonly used form of Farkas' Lemma,
  which easily follows from @{thm [source] Motzkin's_transposition_theorem}. 
  It only permits non-strict inequalities and, as a result, 
  the sum of inequalities will always be non-strict.‚Ä∫

lemma Farkas'_Lemma: fixes cs :: "rat le_constraint set"
  assumes only_non_strict: "lec_rel ` cs ‚äÜ {Leq_Rel}"  
    and fin: "finite cs" 
  shows "(‚àÑ v. ‚àÄ c ‚àà cs. v ‚ä®‚á©l‚á©e c) ‚ü∑ 
       (‚àÉ C const. (‚àÄ (r, c) ‚àà set C. r > 0 ‚àß c ‚àà cs)
         ‚àß (‚àë (r,c) ‚Üê C. Leqc (r *R lec_poly c) (r *R lec_const c)) = Leqc 0 const
         ‚àß const < 0)" 
    (is "_ = ?rhs")
proof -
  {
    fix c
    assume "c ‚àà cs" 
    with only_non_strict have "lec_rel c = Leq_Rel" by auto
    then have "‚àÉ p const. c = Leqc p const" by (cases c, auto)
  } note leqc = this
  let ?lhs = "‚àÉC const rel.
       (‚àÄ(r, c)‚ààset C. 0 < r ‚àß c ‚àà cs) ‚àß
       (‚àë(r, c)‚ÜêC. Le_Constraint (lec_rel c) (r *R lec_poly c) (r *R lec_const c)) 
           = Le_Constraint rel 0 const ‚àß
       (rel = Leq_Rel ‚àß const < 0 ‚à® rel = Lt_Rel ‚àß const ‚â§ 0)" 
  show ?thesis unfolding Motzkin's_transposition_theorem[OF fin]
  proof
    assume ?rhs
    then obtain C const where C: "‚ãÄ r c. (r, c)‚ààset C ‚üπ 0 < r ‚àß c ‚àà cs" and
      sum: "(‚àë(r, c)‚ÜêC. Leqc (r *R lec_poly c) (r *R lec_const c)) = Leqc 0 const"  and
      const: "const < 0" by blast

lemma Farkas'_Lemma_indexed: fixes c :: "nat ‚áí rat le_constraint"
  assumes only_non_strict: "lec_rel ` c ` Is ‚äÜ {Leq_Rel}"  
  and fin: "finite Is" 
  shows "(‚àÑ v. ‚àÄ i ‚àà Is. v ‚ä®‚á©l‚á©e c i) ‚ü∑ 
       (‚àÉ C const. (‚àÄ i ‚àà Is. C i ‚â• 0)
         ‚àß (‚àë i ‚àà Is. Leqc ((C i) *R lec_poly (c i)) ((C i) *R lec_const (c i))) = Leqc 0 const
         ‚àß const < 0)" 
proof -

  have fin: "finite ?C" using fin by auto
  have "(‚àÑ v. ‚àÄ i ‚àà Is. v ‚ä®‚á©l‚á©e c i) = (‚àÑ v. ‚àÄ cc ‚àà ?C. v ‚ä®‚á©l‚á©e cc)" by force
  also have "‚Ä¶ = (‚àÉ C const. (‚àÄ i ‚àà Is. C i ‚â• 0)
         ‚àß (‚àë i ‚àà Is. Leqc ((C i) *R lec_poly (c i)) ((C i) *R lec_const (c i))) = Leqc 0 const
         ‚àß const < 0)" (is "?l = ?r")
  proof
    assume ?r
    then obtain C const where r: "(‚àÄ i ‚àà Is. C i ‚â• 0)" 
         and eq: "(‚àë i ‚àà Is. Leqc ((C i) *R lec_poly (c i)) ((C i) *R lec_const (c i))) = Leqc 0 const" 
         and "const < 0" by auto

      obtain Isl where isl: "set Isl = Is" and dist: "distinct Isl" by auto
    let ?CC = "filter (Œª rc. fst rc ‚â† 0) (map (Œª i. (C i, c i)) Isl)" 
    show ?l unfolding Farkas'_Lemma[OF only_non_strict fin]
    proof (intro exI[of _ ?CC] exI[of _ const] conjI)
      show "const < 0" by fact
      show "‚àÄ (r, ca) ‚àà set ?CC. 0 < r ‚àß ca ‚àà ?C" using r(1) isl by auto
      show "(‚àë(r, c)‚Üê?CC. Le_Constraint Leq_Rel (r *R lec_poly c) (r *R lec_const c)) =
        Le_Constraint Leq_Rel 0 const" unfolding eq[symmetric]
        by (subst sum_list_map_filter, force simp: zero_le_constraint_def,
          unfold map_map o_def, subst sum_list_distinct_conv_sum_set[OF dist], rule sum.cong, auto simp: isl)
    qed
  next
    assume ?l
    from this[unfolded Farkas'_Lemma_set_sum[OF only_non_strict fin]]
    obtain C const where nonneg: "(‚àÄc‚àà ?C. 0 ‚â§ C c)" 
     and sum: "(‚àëc‚àà ?C. Le_Constraint Leq_Rel (C c *R lec_poly c) (C c *R lec_const c)) =
        Le_Constraint Leq_Rel 0 const" 
     and const: "const < 0" 
      by blast
    define I where "I = (Œª i. (C (c i) / rat_of_nat (card (Is ‚à© { j. c i = c j}))))" 
    show ?r
    proof (intro exI[of _ I] exI[of _ const] conjI const)
      show "‚àÄi ‚àà Is. 0 ‚â§ I i" using nonneg unfolding I_def by auto
      show "(‚àë i ‚àà Is. Le_Constraint Leq_Rel (I i *R lec_poly (c i)) (I i *R lec_const (c i))) =
        Le_Constraint Leq_Rel 0 const" unfolding sum[symmetric]
        unfolding sum.image_gen[OF ‚Äπfinite Is‚Ä∫, of _ c]
      proof (rule sum.cong[OF refl], goal_cases)
        case (1 cc)
        define II where "II = (Is ‚à© {j. cc = c j})" 
        from 1 have "II ‚â† {}" unfolding II_def by auto
        moreover have finII: "finite II" using ‚Äπfinite Is‚Ä∫ unfolding II_def by auto
        ultimately have card: "card II ‚â† 0" by auto
        let ?C = "Œª II. rat_of_nat (card II)" 
        define ii where "ii = C cc / rat_of_nat (card II)" 
        have "(‚àëi‚àà{x ‚àà Is. c x = cc}. Le_Constraint Leq_Rel (I i *R lec_poly (c i)) (I i *R lec_const (c i)))
          = (‚àë i‚àà II. Le_Constraint Leq_Rel (ii *R lec_poly cc) (ii *R lec_const cc))"
          unfolding I_def ii_def II_def by (rule sum.cong, auto)
        also have "‚Ä¶ = Le_Constraint Leq_Rel ((?C II * ii) *R lec_poly cc) ((?C II * ii) *R lec_const cc)"

            scaleRat_left_distrib)
        also have "?C II * ii = C cc" unfolding ii_def using card by auto
        finally show ?case .

    qed
  qed
  finally show ?thesis .

text ‚ÄπThis is the main lemma: a finite set of linear constraints is 
  satisfiable over Q if and only if it is satisfiable over R.‚Ä∫
lemma rat_real_conversion: assumes "finite cs" 
  shows "(‚àÉ v :: rat valuation. v ‚ä®‚á©c‚á©s cs) ‚ü∑ (‚àÉ v :: real valuation. v ‚ä®‚á©r‚á©c‚á©s cs)" 
proof
  show "‚àÉv. v ‚ä®‚á©c‚á©s cs ‚üπ ‚àÉv. v ‚ä®‚á©r‚á©c‚á©s cs" using of_rat_val_constraint by auto
  assume "‚àÉv. v ‚ä®‚á©r‚á©c‚á©s cs" 
  then obtain v where *: "v ‚ä®‚á©r‚á©c‚á©s cs" by auto
  show "‚àÉv. v ‚ä®‚á©c‚á©s cs" 
  proof (rule ccontr)
    assume "‚àÑv. v ‚ä®‚á©c‚á©s cs" 
    from farkas_coefficients[OF assms] this
    obtain C where "farkas_coefficients cs C" by auto
    from this[unfolded farkas_coefficients_def]
    obtain d rel where
      isleq: "(‚àÄ(r,c) ‚àà set C. c ‚àà cs ‚àß is_le (r *R c) ‚àß r ‚â† 0)" and
      leq: "(‚àë (r,c) ‚Üê C. lec_of_constraint (r *R c)) = Le_Constraint rel 0 d" and
      choice: "rel = Lt_Rel ‚àß d ‚â§ 0 ‚à® rel = Leq_Rel ‚àß d < 0" by blast
    {
      fix r c
      assume c: "(r,c) ‚àà set C" 
      from c * isleq have "v ‚ä®‚á©c c" by auto
      hence v: "v ‚ä®‚á©c (r *R c)" by (rule sat_scale_rat_real)
      from c isleq have "is_le (r *R c)" by auto
      from lec_of_constraint_real[OF this] v 
      have "v ‚ä®‚á©l‚á©e of_rat_lec (lec_of_constraint (r *R c))" by blast
    } note v = this
    have "Le_Constraint rel 0 (of_rat d) = of_rat_lec (‚àë (r,c) ‚Üê C. lec_of_constraint (r *R c))" 
      unfolding leq by simp
    also have "‚Ä¶ = (‚àë (r,c) ‚Üê C. of_rat_lec (lec_of_constraint (r *R c)))" (is "_ = ?sum")
      unfolding of_rat_lec_sum map_map o_def by (rule arg_cong[of _ _ sum_list], auto)
    finally have leq: "Le_Constraint rel 0 (of_rat d) = ?sum" by simp
    have "v ‚ä®‚á©l‚á©e Le_Constraint rel 0 (of_rat d)" unfolding leq
      by (rule satisfies_sumlist_le_constraints, insert v, auto)
    with choice show False by (auto simp: linear_poly_sum)
  qed
qed

  "simplex_index cs = Sat v ‚üπ ‚ü®v‚ü© ‚ä®‚á©c‚á©s (snd ` set cs)" ‚Äï ‚Äπsatisfying assingment‚Ä∫
  using simplex_index(1)[of cs I] simplex_index(2)[of cs v] 
    rat_real_conversion[of "Simplex.restrict_to (set I) (set cs)"]
  by auto

  "simplex cs = Unsat I ‚üπ ¬¨ (‚àÉ v. v ‚ä®‚á©r‚á©c‚á©s set cs)" ‚Äï ‚Äπunsat of original constraints over the reals‚Ä∫
  "simplex cs = Unsat I ‚üπ set I ‚äÜ {0..<length cs} ‚àß ¬¨ (‚àÉ v. v ‚ä®‚á©r‚á©c‚á©s {cs ! i | i. i ‚àà set I})

  from simplex(2)[OF unsat, unfolded rat_real_conversion[OF this]]
  show "set I ‚äÜ {0..<length cs} ‚àß ¬¨ (‚àÉ v. v ‚ä®‚á©r‚á©c‚á©s {cs ! i | i. i ‚àà set I})
    ‚àß (‚àÄJ‚äÇset I. ‚àÉv. v ‚ä®‚á©c‚á©s {cs ! i |i. i ‚àà J})" by auto
qed (insert simplex(3), auto)
